# Getting Started

Welcome to the project. This guide will walk you through everything you need to know.

## Installation

```bash
pnpm install
```

## Configuration

Copy the example environment file and fill in your values:

```bash
cp .env.example .env
```

## Running the Project

Start the development server:

```bash
pnpm dev
```
